## 需求

1. 主页 
2. 推荐
2. 新品（最近上新）
3. 分类（花色、适用场景、厂商、发布日期、是否推荐）
4. 简介
5. 登录（qq / 微信 / 找回密码（默认邮箱，否则按注册日期））
6. 个人中心（个人信息，邮箱绑定，收藏，浏览历史）
7. 管理员
- 显示：分类页
- 修改分类：可修改分类信息
- 新建分类
- 上传：选择分类后进入后上传
- 管理：删除图片，编辑图片信息
